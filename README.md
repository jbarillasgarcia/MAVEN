## Basic Commands
- sudo apt update
- sudo apt install maven
- mvn -version

## Maven Home
- /opt/maven
or
- /usr/share/maven

## Local Repository
- /home/<username>/.m2
  
## NetBeans 8.2 Setup
- Menu Tools
- > Options
- > Java
- > Maven
- > Category: Execution
- > Maven Home: Browse...

## Manually Index
- Menu Tools
- > Options
- > Java
- > Maven
- > Category: Index
- > Index Update Frequency
- > Click on 'Index Now'
